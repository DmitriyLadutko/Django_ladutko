1. Для работы с проектом выполните команту pip install -r requirements.txt это позволит Вам использовать теже версии и набор утилит, что и при создании проекта
2. для создания миграций моделей используйте команду python (or) python3 manage.py makemigrations
3. для применения миграция используйте команду python (or) python3 manage.py migrate
